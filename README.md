# 🏦 SecureBank - Blockchain-Based Smart Banking System

A modern **web-based banking application** built using **HTML, CSS, and JavaScript** that simulates real-world banking operations with enhanced security features such as PIN authentication, blockchain-inspired transaction records, security token rewards, QR code payments, Fixed Deposits (FD), Recurring Deposits (RD), Loan Management, and Text-to-Speech (TTS).

This project is designed for students to learn banking application development, cybersecurity concepts, and frontend web technologies through a practical implementation.

---

# 📖 Overview

SecureBank is a smart banking simulator that allows users to create accounts, securely log in, manage balances, apply for loans, create deposits, generate QR codes, and maintain transaction history.

The application demonstrates secure authentication, client-side data management using Local Storage, blockchain-inspired transaction logging, and user-friendly banking features in a responsive interface.
